# Beautiful Days - 白金台の癒しエステサロン

**上品で優しい雰囲気のエステサロンサイト**（練習サンプル）

書籍『動くWebデザインアイディア帳』を参考に作成した、白金台にあるプライベートエステサロンのランディングページです。  
ローディング時の光るテキスト、スクロールで背景拡大、メニューカルーセル、ぼかしホバー、ふわっと登場アニメーションなど、癒しと高級感を演出する細やかな動きが特徴です。

## デモ（ライブプレビュー）

## 主な実装機能
- **ローディングアニメーション** - テキストが1文字ずつほのかに光りながら出現
- **ヘッダー背景拡大** - スクロールでゆっくりズームアウト（パララックス）
- **ハンバーガーメニュー** - クリックで背景ぼかし + ナビ出現
- **メニューカルーセル**（slick-carousel） - 4→3→2→1枚のレスポンシブ対応
- **ふわっと登場**（fadeInTrigger） - 各セクションがスクロールで優しく現れる
- **左からスライド登場**（slide-in leftAnime） - 見出しテキスト
- **ホバーで波紋が広がるエフェクト** (circle)  - Menuの各メニュー
- **ページトップボタン** - スクロールで下から出現

## 使用技術・ライブラリ
- HTML5 / CSS3
- jQuery 3.4.1
- [slick-carousel](https://kenwheeler.github.io/slick/) - メニューのスライダー
- Google Fonts（Noto Serif JP + Parisienne）
